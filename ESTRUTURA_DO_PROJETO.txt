╔══════════════════════════════════════════════════════════════╗
║                 FLUXO DE TRABALHO TÍPICO                     ║
╚══════════════════════════════════════════════════════════════╝

CENÁRIO 1: Enviar mensagem secreta
───────────────────────────────────
  1. Use: src/cifra_numerica/codificador_decodificador_numerico.py
  2. Digite sua mensagem
  3. Escolha uma chave (ex: 5)
  4. Envie os números para seu amigo
  5. Compartilhe a chave separadamente

CENÁRIO 2: Receber mensagem secreta
────────────────────────────────────
  1. Use: src/cifra_numerica/decodificador_mensagem_numerica.py
  2. Cole os números recebidos
  3. Digite a chave que seu amigo compartilhou
  4. Veja a mensagem decifrada!

CENÁRIO 3: Quebrar um código (sem chave)
─────────────────────────────────────────
  1. Leia: docs/GUIA_DESCRIPTOGRAFIA.txt
  2. Aprenda sobre força bruta
  3. Teste as 26 possibilidades manualmente
  4. Ou crie um script automatizado!
