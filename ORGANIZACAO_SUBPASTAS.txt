Exemplo: "hello" → "khoor"

Quando usar:
  ✓ Aprender conceitos básicos
  ✓ Criptografia simples de texto
  ✓ Demonstrações educacionais
  ✓ Manter formato de texto

🔢 CIFRA NUMÉRICA
────────────────────────────────────────────────────────────────
Local: src/cifra_numerica/
Tipo: Texto ↔ Números
Exemplo: "ataque" → [22, 15, 22, 12, 16, 26]

Quando usar:
  ✓ Adicionar camada extra de segurança
  ✓ Disfarçar completamente o texto
  ✓ Transmitir como números
  ✓ Combinar múltiplas técnicas

Arquivos:
  • codificador_decodificador_numerico.py (codificar)
  • decodificador_mensagem_numerica.py (decodificar)
