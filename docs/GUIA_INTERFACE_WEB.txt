# 🌐 INTERFACE WEB - GUIA DE USO

## ✨ O que é?

Uma interface web completa e interativa para usar todas as funcionalidades
da Cifra de César diretamente no navegador!

## 🚀 Como Usar

### Método 1: Abrir Localmente
```
1. Navegue até a pasta docs/
2. Clique duas vezes em index.html
3. Abre automaticamente no seu navegador padrão
```

### Método 2: Via Linha de Comando
```bash
# Windows
start docs/index.html

# Linux
xdg-open docs/index.html

# Mac
open docs/index.html
```

## 📋 Funcionalidades Disponíveis

### 🔐 1. Cifrar Tradicional
- Digite uma mensagem
- Escolha uma chave (deslocamento)
- Clique em "Cifrar Mensagem"
- Receba o texto cifrado

**Exemplo:**
```
Mensagem: hello
Chave: 3
Resultado: khoor
```

### 🔓 2. Decifrar Tradicional
- Cole a mensagem cifrada
- Digite a chave usada na cifragem
- Clique em "Decifrar Mensagem"
- Veja o texto original

**Exemplo:**
```
Cifrada: khoor
Chave: 3
Resultado: HELLO
```

### 🔢 3. Codificar em Números
- Digite uma mensagem
- Escolha uma chave
- Clique em "Codificar em Números"
- Receba a sequência numérica

**Exemplo:**
```
Mensagem: ataque
Chave: 5
Resultado: [22, 15, 22, 12, 16, 26]
```

### 🔍 4. Decodificar Números
- Cole os números (separados por vírgula)
- Digite a chave usada
- Clique em "Decodificar Números"
- Veja a mensagem original

**Exemplo:**
```
Números: 22, 15, 22, 12, 16, 26
Chave: 5
Resultado: ATAQUE
```

### 📊 5. Consulta de Alfabeto
- Digite um deslocamento
- Clique em "Gerar Alfabeto"
- Veja o alfabeto deslocado completo
- Consulte letras individuais

**Exemplo:**
```
Deslocamento: 3
Alfabeto: defghijklmnopqrstuvwxyzabc

Consultar: h → Posição 5
```

## 💡 Dicas de Uso

### ✅ Boas Práticas
- Use apenas letras sem acentos (a-z)
- Mantenha a chave entre -25 e 25
- Anote a chave para poder decifrar depois
- Teste com exemplos simples primeiro

### ⚠️ Limitações
- Não suporta caracteres acentuados (á, é, í, ó, ú)
- Espaços e pontuação são mantidos na cifra tradicional
- Espaços são removidos na cifra numérica

## 📖 Exemplos Práticos

### Cenário 1: Enviar Mensagem Secreta
```
1. Abra a aba "Cifrar Tradicional"
2. Digite: "reuniao hoje as 3"
3. Chave: 7
4. Copie o resultado cifrado
5. Envie para seu amigo
6. Compartilhe a chave 7 separadamente
```

### Cenário 2: Receber Mensagem
```
1. Abra a aba "Decifrar Tradicional"
2. Cole a mensagem cifrada recebida
3. Digite a chave que seu amigo compartilhou
4. Clique em decifrar
5. Veja a mensagem original!
```

### Cenário 3: Código Numérico
```
1. Abra "Codificar em Números"
2. Digite sua mensagem
3. Escolha uma chave
4. Envie os números
5. Amigo usa "Decodificar Números" com a mesma chave
```

## 🆘 Solução de Problemas

### Problema: Página não abre
- **Solução:** Verifique se tem um navegador instalado

### Problema: Nada acontece ao clicar
- **Solução:** Aguarde carregar o Brython (CDN)
- **Alternativa:** Verifique conexão com internet

### Problema: Caracteres estranhos
- **Solução:** Não use acentos, apenas a-z

### Problema: Números não decodificam
- **Solução:** Use vírgula entre números: 1, 2, 3
