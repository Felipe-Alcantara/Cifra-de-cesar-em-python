🔐 Cifra de César

Interface Web Interativa — Criptografe e Decifre Mensagens

🔐 Cifrar Mensagem (Tradicional)

ℹ️ Como funciona: Desloca cada letra do alfabeto por N posições. Exemplo: Com deslocamento 3, 'A' vira 'D'.
🌍 Suporte a acentos: Caracteres como á, é, ç são automaticamente convertidos (á→a, ç→c).

🔓 Decifrar Mensagem (Tradicional)

ℹ️ Como funciona: Reverte o processo de cifragem usando a mesma chave.
🌍 Suporte a acentos: Mensagens com acentos são normalizadas automaticamente.

🔢 Codificar em Números

ℹ️ Como funciona: Converte cada letra em um número (1-26) usando alfabeto deslocado. Adiciona camada extra de segurança.
🌍 Suporte completo: Preserva espaços (0), números originais (-0 a -9) e caracteres especiais. Acentos são convertidos (ã→a, ê→e, ç→c).

🔍 Decodificar Números

ℹ️ Como funciona: Converte números de volta para texto usando a chave correta.
💡 Aceita: Números positivos (letras), 0 (espaço), negativos (dígitos e símbolos).

📊 Consulta de Alfabeto Deslocado

ℹ️ Como funciona: Veja como fica o alfabeto com diferentes deslocamentos e consulte posições.
